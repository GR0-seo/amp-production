Extensions that will break with Magento Security Patch SUPEE-6788

In another post we discussed a new Security Patch SUPEE-6788. As we stated in that blog, this important but delicate patch must be implemented correctly, because it will break existing extensions when installed. This release is specifically for Magento Community and Magento Enterprise users.

This patch is a major security fix as it addresses and fixes over 10 security issues, such as information vulnerabilities, and other deep rooted issues in the Magento CMS. Make sure you’ve implemented all previous patches to your site, as this one will only work properly after all previous ones have been installed. You can find them here: http://magento.com/security/patches

Once you are certain all previous patches are in place, you will want to carefully check the following items to make sure your site does not crash when you install the SUPEE-6788 security patch.
